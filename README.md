## 🛠️ 如何选择适合你的机场？

1.  **看协议：** 优先选择 **Shadowsocks (SS)** 或 Trojan，兼容性强且性能开销小。
2.  **看线路：** 尽量选择带 **“中转”** 或 **“专线 (IEPL/IPLC)”** 字样的节点，稳定性远超直连。
3.  **看售后：** 优质机场通常拥有活跃的 Telegram 频道或工单系统。


3. 请遵守当地法律法规，严禁用于任何非法用途。
